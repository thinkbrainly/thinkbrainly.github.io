4. if a house is 30 ft wide and 45 ft long on a scale drawing the house is 4 in wide & 6 in long, a bedroom is 1.2 in. wide and 1.6 in. long what are the actual measurements of this bedroom


Answer:

The answer to this problem are width = 9 ft and length = 12 ft.

Step-by-step explanation:

This is a type of problem that deals with ratio and proportion.

First, we need to solve the relationship of the actual width and length of the house to the scaled measurements.

The solution goes like this

Let x = proportionality constant for width

and y = proportionality constant for length

Width

[tex]30 = 4x\\x = \frac{30}{4}\\x = 7.5[/tex]

Thus the actual width of the room is

[tex]W = 1.2(7.5) =9 ft[/tex]

Length

[tex]45 = 6y\\y = \frac{45}{6}\\y = 7.5[/tex]

Thus the actual length of the room is

[tex]W = 1.6(7.5) =12 ft[/tex]

28. A housing project has 1822 units. There are twice as many two-bedroom units as there are one-bedroom units, and 250 fewer three-bedroom units than one-bedroom units. How many of each kind of units are there?with full solution pls ​


Let's use some variables to represent the number of each type of unit:

Let x be the number of one-bedroom units.

Then, the number of two-bedroom units would be 2x (twice as many as one-bedroom units).

And, the number of three-bedroom units would be x - 250 (250 fewer than one-bedroom units).

The total number of units is 1822, so we can write an equation:

x + 2x + (x - 250) = 1822

Simplifying and solving for x:

4x - 250 = 1822

4x = 2072

x = 518

So there are 518 one-bedroom units, 2x = 1036 two-bedroom units, and x - 250 = 268 three-bedroom units.

Therefore, there are 518 one-bedroom units, 1036 two-bedroom units, and 268 three-bedroom units.

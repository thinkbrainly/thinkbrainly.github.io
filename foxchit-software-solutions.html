4. a combination of operating system utility programs and applications are essential to the computer which of the following is being described based on the statementA. Application softwareB.Software packagesC. Security softwareD.System software​


Answer:

D. System Software

Explanation:

System software is required in order for a PC to boot Windows, MacOS, Linux, etc.

Reasoning:

Why not A? Application software is a software that accomplishes certain tasks for the user. Examples of application software is Photoshop, MS Word, etc. It is not related to a computer's ability to boot and function.

Why not B? Software packages is a bundle of software designed to work together. One example of this is Microsoft Office consisting of MS Word, MS Powerpoint, MS Excel, etc. It is completely not related to a computer's essentials in order to boot.

Why not C? Security Software is a software that is used in order to protect the data that can be accessed by users with differing levels of authority. This can also protect from malicious attacks i.e viruses. It does not have any effects on booting up a computer.

Mathematical programming refers to mathematical models used to solve problems such as decision problems. The terms are meant to contrast with computer programming which solves such problems by implementing algorithms which may be designed specifically for a given problem. By mathematical programming, we consider declarative approaches. This means that a separation is considered between the representation of the problem through a mathematical model and its solving. The idea is that solving may be done through general methods, such as branching methods, using the mathematical model designed to capture the problem. For instance, considering a decision problem that can be represented by a graph, variables can represent the presence or absence of such and such vertices and edges in the solution.

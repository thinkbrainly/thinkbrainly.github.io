Evaluate the ages of Lara and Lara's father in an algebraic sentence

Lara's age = x

Lara's father's age = 4x + 16

So the expression would be

4x + 16 = 36, find x

4x + 16 = 36

4x = 36 - 16

4x = 20

x = 5

Checking

4x + 16 = 36

4(5) + 16 = 36

20 + 16 = 36

36 = 36 -- true

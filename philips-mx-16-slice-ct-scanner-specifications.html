4. Find the indicated measures. Use the figure on the right.19. If CT=LT, GC=3x+7 units, and EL=16,then. x=____20. If GR=EA,CT=2x+9, and TL=4x-7,then CT_____21. If CT=LT, GR=x^2+2x+6, and EA =x^2+4x-10,then EL=____​


Answer:

19. Since CT=LT, we know that GC+CL=GL which means that 3x+7+CL=16. Solving for CL, we get CL=9-3x. But we also know that CT=LT, so CL+TL=CT which means (9-3x)+(4x-7)=2x+9. Solving for x, we get x=5.

20. Since GR=EA, we know that GC+CR=EL which means (2x+9)+(4x-7)=16. Solving for x, we get x=2. Therefore, CT=2(2)+9=13 units.

21. Since CT=LT, we know that GC+CL=GL which means that x^2+2x+6+(4x-7)=x^2+4x-4. Solving for x, we get x=5. Then, EL=GC+CL=5(5)+7=32 units.

2. A sample of 0?1350g of As203 was assayed iodometrically using 23.4 ml of 0.1055N 12 solution. What is the %P of the sample​


Answer:

Explanation:

To calculate the percentage of As₂O₃ in the sample, we need to use the following formula:

% As₂O₃ = (B/A) x (N x 197) x 100

Where,

B = volume of iodine solution used (in liters)

A = weight of sample used (in grams)

N = normality of iodine solution

197 = molecular weight of As₂O₃

Given:

Weight of sample (A) = 0.1350 g

Volume of iodine solution (B) = 23.4 ml = 0.0234 L

Normality of iodine solution (N) = 0.1055 N

Substituting the values in the formula, we get:

% As₂O₃ = (0.0234/0.1350) x (0.1055 x 197) x 100

% As₂O₃ = 75.04%

Therefore, the percentage of As₂O₃ in the sample is 75.04%.
